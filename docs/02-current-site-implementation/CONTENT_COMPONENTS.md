# Content Components

## Goal

Identify reusable content components that support case-study-first storytelling in the current Astro site.

## Current Components

- `ProjectCard`
- `CaseMeta`
- `FlowDiagram`
- `MediaFull`
- `MediaTwo`
- `MediaThree`
- `Section`

## Near-Term Component Needs

- Featured case study card
- Status badge for in-progress case studies
- Outcome metric card
- Reflection block
- Prototype/video embed wrapper
- Lab project card
- Writing card

## Implementation Guidance

- Prefer extending current components before adding new abstractions.
- Keep MDX authoring simple.
- Make captions and metadata do more storytelling work.
- Use semantic HTML and accessible media labels.
