# Accessibility And Performance Checklist

## Accessibility

- [ ] Semantic landmarks are present.
- [ ] Heading order is logical.
- [ ] Images have useful alt text.
- [ ] Video embeds have accessible labels or captions where needed.
- [ ] Focus states are visible.
- [ ] Keyboard navigation works.
- [ ] Color contrast is sufficient.
- [ ] Motion respects reduced-motion preferences.

## Performance

- [ ] Images are optimized.
- [ ] Videos are compressed and lazy where possible.
- [ ] Embeds are lazy-loaded.
- [ ] JavaScript is avoided unless needed.
- [ ] Homepage remains fast.
- [ ] Build completes successfully.
