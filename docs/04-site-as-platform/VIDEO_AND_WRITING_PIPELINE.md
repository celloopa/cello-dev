# Video And Writing Pipeline

## Goal

Connect portfolio work to YouTube, LinkedIn, writing, and process notes.

## Content Flow

```txt
Project work -> Process notes -> Case study -> Video walkthrough -> LinkedIn posts -> Resume/GitHub proof
```

## Writing Types

- Product design notes
- Design-dev collaboration notes
- Prototype build logs
- Data model explanations
- Case study reflections
- Accessibility and implementation notes

## Video Types

- Case study walkthroughs
- Prototype demos
- Build logs
- Design-dev notes
- Portfolio rebuild updates

## Publishing Rule

Every major public launch should have:

- One portfolio page
- One short LinkedIn post
- Optional longer process note
- Optional video walkthrough
- GitHub README update if there is code
